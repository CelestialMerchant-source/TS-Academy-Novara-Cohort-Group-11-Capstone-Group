 🪐 TS Academy Novara Cohort — Group 11 Capstone Project

A React web application that explores our Solar System through data.
Built collaboratively by Group 11 of the TS Academy Novara Cohort.
The app fetches real planet data from an external API and presents it
in a clean, responsive, and interactive interface.

 ✅ Features

- 🪐 Planet data dynamically fetched from external API
- 📱 Fully responsive — mobile and desktop layouts
- 🎥 Autoplay, muted, looping background video (no iframe)
- 🔗 Smooth scroll on "Explore the Data" and "Contact Us" buttons
- 📩 Contact form with client-side validation and success message
- 📊 Comparative planetary data table



 🛠️ Tech Stack

- React (Vite)
- HTML5 & CSS3
- JavaScript ES6+
- Fetch API
- Git & GitHub
- Netlify / Vercel (Deployment)



 ⚙️ How to Run Locally

```bash
 1. Clone the repository
git clone https://github.com/CelestialMerchant-source/TS-Academy-Novara-Cohort-Group-11-Capstone-Group.git

 2. Navigate into the project folder
cd TS-Academy-Novara-Cohort-Group-11-Capstone-Group

 3. Install dependencies
npm install

 4. Start the development server
npm run dev
```

Then open your browser and go to `http://localhost:5173`


🌿 Git Workflow

1. All work is done on separate branches — never directly on `main.`
2. Every contribution is submitted via a Pull Request
3. The person who opens a PR cannot review or merge their own PR
4. All PRs must be reviewed and approved before merging into `main.`
5. Branch naming format: `feature/component-name` (e.g. `feature/contact-form`)
